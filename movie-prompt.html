🎬 영화/애니 프롬프트 생성기
영상용 메인 프롬프트 · 4컷 콘티 · 3x3 스타일 테스트 · SRT 자막 분리 · by 제일라
이제 출력 구조가 이렇게 바뀜:
1) 영상용 메인 프롬프트 → Veo / Flow / Kling 같은 영상툴에 넣는 용도
2) 4컷 콘티 프롬프트 → 이미지 생성용 콘티 확인용
3) 3x3 스타일 테스트 프롬프트 → 이미지 생성용 캐릭터/스타일 점검용
4) SRT 자막 → 영상 생성 후 편집툴에서 따로 넣는 용도

중요: 4컷 콘티, 3x3 그리드는 영상 생성 툴에 넣는 게 아님. 영상 만들 때는 씬별 "영상용 메인 프롬프트"만 사용.
🔑 API KEY
OpenAI API 키를 입력하면 AI 대본 생성 / 캐릭터 분석이 활성화됩니다.
🔒 API 키는 이 브라우저 localStorage에 저장됩니다. 공용 PC에서는 사용 후 반드시 삭제하세요.
