🔑 API KEY
OpenAI API 키를 입력하면 캐릭터 분석 / AI 포즈 추천이 활성화됩니다. (이미지 생성은 사용자가 직접 Midjourney 등에 프롬프트 던지기)
🔒 API 키는 이 브라우저 localStorage에 저장됩니다. AI 호출 시에는 프록시 서버 또는 OpenAI API로 요청에 포함되어 전송됩니다. 공용 PC에서는 사용 후 반드시 삭제해주세요.
📁 프로젝트 (캐릭터별 폴더)
한 캐릭터를 하나의 프로젝트로 관리. 같은 캐릭터에 무한히 포즈를 추가할 수 있어요.
아직 프로젝트가 없어요. + 새 프로젝트 눌러서 첫 캐릭터를 만드세요.
